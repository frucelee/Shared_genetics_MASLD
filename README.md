# Shared_genetics_MASLD
This repository contains the code used for the analyses presented in the publication "CTCF-rs705704-SUOX axis is important for the association between hypothyroidism and metabolic dysfunction-associated steatotic liver disease".

# Citation
Li S, Chen D, Gong M. CTCF-rs705704-SUOX axis is important for the association between hypothyroidism and metabolic dysfunction-associated steatotic liver disease. Gut 2026. DOI: 10.1136/gutjnl-2026-339457.
