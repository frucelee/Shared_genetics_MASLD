# Shared_genetics_MASLD
This repository contains the code used for the analyses presented in the publication "The CTCF-rs705704-SUOX axis is important for the association between hypothyroidism and metabolic dysfunction-associated steatotic liver disease".

# Citation
Li SF, Chen DW, Gong MJ. The CTCF-rs705704-SUOX axis is important for the association between hypothyroidism and metabolic dysfunction-associated steatotic liver disease. Gut 2026. http://dx.doi.org/10.1136/gutjnl-2026-339457
